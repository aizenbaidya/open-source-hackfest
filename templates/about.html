Welcome to Recall Royale!

Test your skills with our interactive flashcard game! Start with easy problems and work your way up to more challenging ones. Can you conquer all the levels?


How to Play

Each flashcard will present a problem from the chosen dataset. Solve enough correctly to move on to the next one. As you progress, the difficulty increases!

Once you conquered a level you can move onto the next stage

There will be 3 levels you will need to pass through in order to complete a topic


About this Project

This project was created for the MLH Open Source Hackfest.

The dataset can be of any topic so it doesn't need to just be programming related topics

Select from one of our pre-made datasets or choose one created by your peers

Help teach others to code by uploading your own dataset of problems
